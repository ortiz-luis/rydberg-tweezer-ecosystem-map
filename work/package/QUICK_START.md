# Quick start for Codex

1. Unzip this package.
2. Create or open the GitHub repository where the website should live.
3. Copy:
   - `data/rydberg_tweezer_ecosystem_map_export_v2_1.geojson`
   to:
   - `public/data/rydberg_tweezer_ecosystem_map_export_v2_1.geojson`
4. Paste the content of `codex_prompt_full_github_pages.txt` into Codex.
5. Let Codex create the React/Vite/Tailwind/Leaflet app.
6. Ask Codex to run:
   - `npm install`
   - `npm run build`
7. Ask Codex to configure GitHub Pages deployment.
